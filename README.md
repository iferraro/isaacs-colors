# Isaac's Custom Colors
### A collection of colors that I've used in personal projects or spotted elsewhere
This is the third iteration of one of my first web projects. 
At the time of each color's discovery or first application, I gave it a name that I found fitting. In this version, I replicated the appearance of a LaTeX document, even to the extent of using Computer Modern font. Questions and feedback on this site are welcome! My contact info is at the site's footer as well as my [profile page](https://github.com/iferraro).
## Link to Site
https://www.isaacscolors.io/
